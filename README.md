### 功能

- 确实：当群友发言中带有“**确实**”这两个字时，小月也会回复一个确实避免冷场。
- 骂鱼哥：发送**骂鱼哥**时，会骂鱼哥
- 揍傻6：发送揍+@傻6时，会揍傻6
- 教小月：作为一名高质量人工智能，自然是要会学习的。发送`教小月 问题 答案`可以教小月
- sur色图：会发sur的色图。
- 转账：发送d@LightL 1000,会给LightL转账1000黄金
